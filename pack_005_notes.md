# Watchman Finance Pack 005 Notes

Included:
- leave types
- leave policies
- leave policy assignments
- leave accrual rules
- employee leave profiles
- leave balance ledger
- leave adjustments
- leave requests
- leave request day breakdown
- leave approvals
- holiday calendars
- leave liability snapshots
- starter select-only RLS policies
- starter leave and accrual server actions

What is intentionally simplified in this pack:
- accrual engine is basic and policy-rate driven
- blackout dates are not yet modeled
- negative balance enforcement is not yet fully implemented
- payout rules are not yet modeled
- leave sync back to Operations is deferred
- leave-to-payroll posting automation is deferred
